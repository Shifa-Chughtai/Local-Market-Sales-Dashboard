# Local Market Sales Dashboard

## Overview

This project is an interactive Power BI dashboard designed to analyze local sales performance. It provides insights into  profit, product categories and daily sales performance.

## Features

Interactive filters and slicers 

Sales trend analysis

Top-performing products

Profit and Sales KPIs

## Tools Used

Power BI

DAX

Power Query

Excel (data source)

## File

Sales Dashboard.pbix – Main Power BI dashboard file

## Purpose

The dashboard helps businesses monitor sales performance, identify trends, and make data-driven decisions through interactive visualizations.
